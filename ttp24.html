Rusthound? Yes!

Description

RustHound is an open-source offensive security tool designed for Active Directory (AD) enumeration and BloodHound data collection, but written in Rust instead of other languages like PowerShell or C#


Why RustHound?

  1. It has been found that many EDRs do not flag RustHound as malicious, which allows the tool to run without generating alerts. This does not detract from the fact that it generates a lot of telemetry which makes it very noisy.
